# 🔥 Roast My Portfolio

An AI-powered developer portfolio roaster built with React and the Groq API.

Paste your portfolio content, choose your roast intensity, and get brutally honest (but helpful) feedback.

---

## Features

- 3 roast levels: Mild, Medium, Savage
- Structured feedback: verdict, roast, real issues, positives, and fixes
- Animated loading skeleton while waiting
- Share your roast on X (Twitter) or copy to clipboard
- Dark theme, fully responsive

---

## Stack

- React (Create React App)
- Groq API (claude-sonnet-4)
- Plain CSS with CSS variables
